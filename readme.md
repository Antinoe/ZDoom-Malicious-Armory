# Malicious Armory

This is a weapon pack for Blue Savior's Project Malice, using Dwelling Sin and Sin Survival as base resources.

### Sidearms
- Pistols (Colt M1911, )
- SMGs

### Longarms
- Shotguns (Shotgun, Flak Cannon)
- Chainguns (Heavy Carbine, Autocannon)
